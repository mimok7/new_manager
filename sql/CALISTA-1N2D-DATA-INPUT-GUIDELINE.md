## 4. 요금 체계

### **A. 성인 요금 (price_adult)**

- **적용**: 18세 이상 고객
- **특징**: 2박3일 대비 약 50% 수준

---

### **B. 아동 요금 (price_child)**

- **나이**: 4세 이상 14세 미만
- **특징**: **모든 객실 동일 요금** ✅

- **구성**: 침구류, 식사 포함

## 7. 데이터 검증 패턴

### **7.1 1N2D 아동 요금 검증**
```sql
SELECT DISTINCT price_child 
FROM cruise_rate_card 
WHERE cruise_name = '칼리스타 크루즈' 
  AND schedule_type = '1N2D'
  AND price_child > 0;
-- 예상 결과: 1행 (2775000)
```

### **7.2 1N2D 2번째 유아 요금 검증**
```sql
SELECT DISTINCT price_infant 
FROM cruise_rate_card 
WHERE cruise_name = '칼리스타 크루즈' 
  AND schedule_type = '1N2D'
  AND price_infant > 0;
-- 예상 결과: 1행 (2950000)
```

### **7.3 1N2D 총 개수 검증 (2N3D 영향 없음)**
```sql
SELECT schedule_type, COUNT(*) as row_count
FROM cruise_rate_card 
WHERE cruise_name = '칼리스타 크루즈'
GROUP BY schedule_type;
-- 예상 결과: 
--   1N2D: 18행
--   2N3D: 18행 (영향 없음)
```

### **7.4 2박3일 보존 확인**
```sql
SELECT COUNT(*) as "2N3D데이터"
FROM cruise_rate_card 
WHERE cruise_name = '칼리스타 크루즈' 
  AND schedule_type = '2N3D';
-- 예상 결과: 18 (변경 없음)
```

### **7.5 1N2D vs 2N3D 가격 비교**
```sql
SELECT 
  s1.room_type as "객실명",
  s1.price_adult as "1N2D_성인",
  s2.price_adult as "2N3D_성인",
  ROUND(s1.price_adult / s2.price_adult * 100, 1) as "비율_%"
FROM (
  SELECT room_type, price_adult FROM cruise_rate_card 
  WHERE cruise_name = '칼리스타 크루즈' AND schedule_type = '1N2D' AND valid_from = '2026-01-01'
) s1
FULL JOIN (
  SELECT room_type, price_adult FROM cruise_rate_card 
  WHERE cruise_name = '칼리스타 크루즈' AND schedule_type = '2N3D' AND valid_from = '2026-01-01'
) s2 ON s1.room_type = s2.room_type
ORDER BY s2.room_type;
-- 예상 결과: 1N2D는 2N3D의 약 50% 수준
```

## 8. 특수 사항

### **8.1 President Suite 아동 요금 처리**
- **아동 요금**: 2,775,000동 (다른 객실과 동일)
- **엑스트라베드**: 불가능 (NULL)
- **아동+엑스트라**: 불가능 (NULL)
- **이유**: 최고급 객실로서 별도 침실 제공

### **8.2 아동 요금 통일 정책**
- **모든 객실**: 2,775,000동 (동일)
- **아동+엑스트라**: 3,825,000동 (동일)
- **2번째 유아**: 2,950,000동 (동일)

### **8.3 schedule_type 중요성**
- 1N2D와 2N3D는 **완전히 별도 상품**
- 데이터 삭제 시 `schedule_type = '1N2D'` 명시 필수
- 2N3D 데이터는 절대 영향받지 않음
