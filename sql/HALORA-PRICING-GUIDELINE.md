## 데이터베이스 적용 규칙

### cruise_rate_card 테이블 구조 (컬럼별 가격 저장)
```
- cruise_name: '할로라 크루즈'
- schedule_type: '1N2D'
- room_type: 객실명 (한글)
- room_type_en: 객실명 (영문)
- price_adult: 성인 요금
- price_child: 아동(8-11세) 요금
- price_infant: 아동(5-7세) 요금
- price_extra_bed: 엑스트라베드 요금 (불가시 NULL)
- price_child_extra_bed: NULL (미사용)
- price_single: 싱글차지 (불가시 NULL)
- extra_bed_available: 엑스트라베드 가능 여부 (true/false)
- valid_year: 2026
- currency: 'VND'
- is_active: true
- valid_from: YYYY-MM-DD
- valid_to: YYYY-MM-DD
- notes: 아동 가격 설명 및 제약사항
```

### 데이터 저장 패턴
각 객실별로 **1개 행** 생성 (시즌별):
- **room_type** + **valid_from** 조합이 유니크 키
- 모든 카테고리 가격을 한 행의 컬럼에 저장
- notes에 아동(5-7세/8-11세) 세부 정보 기록
