## 4. 요금 체계

### **A. 성인 요금 (price_adult)**
- **정의**: 1명의 성인이 점유할 경우의 기본 요금
- **적용**: 18세 이상 고객
- **특징**: 시즌별로 변동
  - Season 1 & 3: 동일 수준 (고가기)
  - Season 2: 14-18% 할인

---

### **B. 아동 요금 (price_child)**
- **정의**: 1명의 아동이 성인과 함께 점유할 경우의 요금
- **나이**: 4세 이상 14세 미만 (정책 기준)
- **특징**: **모든 객실 동일 요금** ✅
  - Season 1, 2, 3: **5,550,000동 고정**
  - President Suite도 동일 적용
- **구성**: 침구류, 식사 포함

---

### **C. 2번째 유아 요금 (price_infant)**
- **정의**: 객실당 2명 이상의 유아가 머물 경우 2번째 유아부터 청구
- **나이**: 1세 ~ 4세
- **요금**: **5,900,000동/인** (모든 객실 동일)
- **정책**:
  - 첫 번째 유아: 무료 ✅
  - 두 번째 유아: 5,900,000동
  - 세 번째 이상: 5,900,000동 × 명수
  - 4세 이상으로 판정되면 아동 요금 적용

---

- **가용성**:
  - **5개 객실**: 가능 ✅

## 7. 데이터 검증 패턴

### **7.1 아동 요금 통일성 검증**
```sql
-- 모든 객실의 아동 요금은 5,550,000 이어야 함
SELECT DISTINCT price_child 
FROM cruise_rate_card 
WHERE cruise_name = '칼리스타 크루즈' 
  AND price_child > 0;
-- 예상 결과: 1행 (5550000)
```

### **7.2 2번째 유아 요금 검증**
```sql
-- 모든 객실의 2번째 유아 요금은 5,900,000 이어야 함
SELECT DISTINCT price_infant 
FROM cruise_rate_card 
WHERE cruise_name = '칼리스타 크루즈' 
  AND price_infant > 0;
-- 예상 결과: 1행 (5900000)
```

### **7.3 President Suite 엑스트라베드 검증**
```sql
-- President Suite만 엑스트라베드가 NULL 이어야 함
SELECT room_type, extra_bed_available, price_extra_bed, price_child_extra_bed
FROM cruise_rate_card 
WHERE cruise_name = '칼리스타 크루즈' 
  AND room_type = 'President Suite';
-- 예상 결과: 
--   extra_bed_available = false
--   price_extra_bed = NULL
--   price_child_extra_bed = NULL
```

### **7.4 시즌별 객실 개수 검증**
```sql
-- 각 시즌마다 6개 객실이 있어야 함
SELECT valid_from, COUNT(*) as row_count
FROM cruise_rate_card 
WHERE cruise_name = '칼리스타 크루즈'

-- 예상 결과: 3행 (각각 6행씩)
```

### **7.5 휴일 요금 검증**

## 8. 특수 사항

### **8.1 President Suite 엑스트라베드 불가 처리**
- **이유**: 최상급 스위트는 별도 침실을 제공하므로 추가 침대 불필요
- **DB 처리**: `extra_bed_available = false`, 가격 필드는 `NULL`
- **고객 안내**:
  - 추가 인원: 인접 객실 업그레이드 권유
  - 아동 동반: 메이크업 서비스(침구 추가 배치) 무료 제공

### **8.2 아동 요금 통일 정책**
- **이유**: 모든 객실의 운영 비용 대비 아동 동반 비용이 동일
- **예외**: President Suite도 동일 아동 요금 (5,550,000) 적용
- **변수**: 엑스트라베드 사용 여부만 가격 변동 (7,650,000)

### **8.3 시즌 2 (프로모션) 엑스트라베드 할인**
- **대상**: 칼리스타 오션, 칼리스타 베이만 적용
- **할인율**: 약 57% 할인 (기본 가격의 50%)
  - 칼리스타 오션: 9,300,000 → 3,975,000
  - 칼리스타 베이: 10,000,000 → 3,975,000
- **이유**: 여름 시즌 가족 단위 수요 촉진
