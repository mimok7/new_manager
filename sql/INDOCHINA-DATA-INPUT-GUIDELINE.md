## 📊 DB 입력 매핑 규칙

### 컬럼 대응 관계

| 컬럼명 | 값 | 설명 |
|--------|-----|------|
| **cruise_name** | 인도차이나 크루즈 | 고정값 |
| **schedule_type** | 1N2D | 1박2일 고정 |
| **room_type** | (객실명) | 주니어/스위트/이그제큐티브/프레지던트 |
| **room_type_en** | (영문명) | Junior Suite / Suite / Executive Suite / President Suite |
| **price_adult** | (성인 가격) | 표에서 성인 컬럼값 |
| **price_child** | (아동 가격) | 표에서 아동(5~11세) 컬럼값 |
| **price_infant** | (유아 가격) | 표에서 유아(4세이하) 컬럼값 |
| **price_extra_bed** | (엑스트라베드) | 표에서 엑스트라베드 컬럼값 |
| **price_child_extra_bed** | (유아 가격) | price_infant과 동일 (4,250,000 또는 4,050,000) |
| **price_single** | (싱글차지) | 표에서 싱글차지 컬럼값 |
| **extra_bed_available** | true | 모든 객실 가능 |
| **valid_year** | 2026 | 고정값 |
| **valid_from** | 2026-05-01 (S1) / 2026-10-01 (S2) | 시즌 시작일 |
| **valid_to** | 2026-09-30 (S1) / 2026-12-31 (S2) | 시즌 종료일 |
| **currency** | VND | 베트남 동 |
| **season_name** | 신용카드 / VND 송금 | 결제방식 |
| **is_active** | true | 활성 상태 |

## ✅ 검증 체크리스트

- [ ] 모든 객실 4개 × 2시즌 × 2결제방식 = 16행 생성됨
- [ ] 신용카드 결제: Season 1 아동가 = 3,400,000
- [ ] VND 송금: Season 1 아동가 = 3,300,000
- [ ] 모든 엑스트라베드: 신용카드 4,200,000 / VND 4,050,000
- [ ] 유아 가격 = 엑스트라베드 가격 (동일)
- [ ] valid_from/valid_to: Season 1 (05/01~09/30), Season 2 (10/01~12/31)
- [ ] season_name: "신용카드" 또는 "VND 송금" (정확한 문자열)
