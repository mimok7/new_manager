- **크루즈명**: 파라다이스 레거시 크루즈 (Paradise Legacy Cruise)

## 데이터베이스 적용 규칙

### cruise_rate_card 테이블 구조 (컬럼별 가격 저장)
```
- cruise_name: '파라다이스 레거시 크루즈'
- schedule_type: '1N2D'
- room_type: 객실명 (한글)
- room_type_en: 객실명 (영문)
- price_adult: 성인 요금
- price_child: 아동 요금 (5-12세)
- price_infant: 유아 요금 (2인째 이후)
- price_extra_bed: 엑스트라베드 요금 (성인용)
- price_child_extra_bed: 아동 엑스트라베드 요금
- price_single: 싱글차지
- extra_bed_available: 엑스트라베드 가능 여부 (true)
- valid_year: 2026
- currency: 'VND'
- is_active: true
- valid_from: YYYY-MM-DD
- valid_to: YYYY-MM-DD
```

### 데이터 저장 패턴
각 객실별로 **1개 행** 생성 (시즌별):
- **room_type** + **valid_from** 조합이 유니크 키
- 모든 카테고리 가격을 한 행의 컬럼에 저장
