## 🔍 기존 앱 계산 방식 (updateQuoteItemPrices.ts)

### 공항 (Airport)
```typescript
// 기존 방식
const { data: priceData } = await supabase
  .from('airport_price')
  .select('price')
  .eq('airport_code', airportData.airport_code);

unitPrice = (priceData[0].price || 0) * (airportData.passenger_count || 1);
```

**신규 중앙 계산기와 동일:**
```typescript

.select('airport_code, service_type, price')
.eq('airport_code', input.airport_code)
const subtotal = unit_price * quantity;  // quantity = passenger_count
```

✅ **호환성:** 100% (공식 동일)

### 호텔 (Hotel)
```typescript
// 기존 방식
const { data: priceData } = await supabase
  .from('hotel_price')
  .select('base_price')
  .eq('hotel_price_code', hotelData.hotel_code);

unitPrice = priceData[0].base_price || 0;  // 박수 미적용!
```

**신규 중앙 계산기 (개선):**
```typescript
// 신규 방식
.select('base_price')
.eq('hotel_price_code', input.hotel_price_code)
const subtotal = base_price * nights * room_count;  // 박수 × 객실 고려
```

⚠️ **호환성:** 쿼리는 호환, 계산 방식은 다름
- 기존: base_price만 사용 (1박 요금)
- 신규: base_price × nights × rooms (전체 요금) ← **더 정확**

**권장:** 신규 중앙 계산기 사용으로 박수 적용

- 기존: price만 사용 (단일 대 요금)
- 신규: price × quantity (여러 대 고려) ← **더 정확**

**권장:** 신규 중앙 계산기 사용으로 quantity 적용

### 투어 (Tour)
```typescript
// 기존 방식
const { data: priceData } = await supabase
  .from('tour_pricing')
  .select('price_per_person')
  .eq('pricing_id', tourData.tour_code);

unitPrice = (priceData[0].price_per_person || 0) * (tourData.participant_count || 1);
```

**신규 중앙 계산기 (개선 예정):**
```typescript
// 신규 방식
.select('tour_code, pricing_id, adult_price, child_price, price_per_person')
const subtotal = adult_price * adult_count + child_price * child_count;
// 또는 adult_price/child_price 없으면 price_per_person 대체
```

✅ **호환성:** 100% (기본 공식 호환)
- 기존: price_per_person × count
- 신규: adult_price × adult + child_price × child
- **개선:** 성인/아동 가격 구분 가능
